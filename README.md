# Spam Email Classifier

## Overview

This project is a Machine Learning based Spam Email Classifier that predicts whether an Email or SMS message is Spam or Ham (Not Spam).

## Technologies

- Python
- Pandas
- NumPy
- NLTK
- Scikit-Learn
- Streamlit

## Machine Learning Algorithm

- TF-IDF Vectorizer
- Multinomial Naive Bayes

## Features

- Spam Detection
- Text Preprocessing
- TF-IDF Feature Extraction
- Real-time Prediction
- Streamlit User Interface

## Run

Install packages

```bash
pip install -r requirements.txt
```

Train Model

```bash
python train_model.py
```

Run Application

```bash
streamlit run app.py
```
